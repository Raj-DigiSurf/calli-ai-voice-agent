The Technology

Built for Kitomba — without the API

Kitomba has no public API. That's exactly why this works.

Every other booking solution needs an API integration. Kitomba doesn't have one. Calli navigates your Kitomba booking page like a human would — opening the browser, reading the live calendar, selecting the slot.

No third-party integrations. No changes to your existing setup. It just works.

  • Reads your real-time Kitomba availability
  • Works with your existing stylists and services
  • No changes to your Kitomba account needed
  • Handles the Facebook login step via SMS link
  • Deposit secured before caller hangs up
